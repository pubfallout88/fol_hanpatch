폴런던 LondonWorldSpace.esm 파일 번역기에 올려서 한방에 번역할 목적으로 직접 제작한 스크립트임.
실제 ESM을 최종 완성하는 것은 xTranslator에서 작업하지만, 번역 API 사용 시 과금 문제 때문에 문서 파일을 추출해서
직접 올리는 것이 유리하다고 판단해서 텍스트 추출 및 텍스트 입력 목적으로 제작한 스크립트임

파이썬으로 제작되어서 파이썬이 설치되어 있어야 하며 (파이썬은 컴퓨터 언어다)
xTranslator에서 XML을 추출해서 번역용 Text 파일을 제작하고 다시 번역된 Text 파일을 XML 파일로 재가공해서
다시 xTranslator 넣기 위한 스크립트 모음으로 되어 있음...

설치법 및 사용법)

일단 파이썬을 설치한다. 
윈도우키 + R 누르면 나오는 창에서 cmd 라고 엔터 치면 명령 프롬프트란 것이 나오고 python 이라고 입력하고 엔터치면
파이썬이 설치할 수 있는 화면이 나온다...파이썬 설치는 생각보다 간단하다.

xTranslator 설치...이건 설명하지 않겠다...
다만, 진짜 중요한 건 xTranslator\Data\Fallout4\codepage.txt 파일 열어서
en=utf8,1252 이거 반드시 해두라...이거 안 하면 라틴어 단어 다 깨진다.

그리고 스크립트 파일 설명 들어간다.

다른 건 필요 없다...상황에 맞게 .bat 배치 파일만 클릭하면 된다.

[1.XML에서번역할텍스트파일추출.bat] --> XML에서 번역용 파일을 추출하는 건데, 번역기용문장.txt이란 것이 자동으로 만들어진다.

이걸 번역기에 문서 파일로 올리면 됨...태그 같은 것도 용량 먹기 때문에 최소한으로 파일 줄여서 만든 거다.

이렇게 번역기용문장.txt 이랑 번역된문장.txt(이건 번역사이트에서 내려받은 거) 이렇게 두개가 생겼다면

[2.번역기에서나온파일을마스터파일로변환.bat] 이거 누르면 편하게 수정할 수 있는 마스터 파일이 생긴다.

마스터 파일은 이미 만들어져있다. 그래서 위에 거 안 해도 됨...

마스터 파일은 2개다...
마스터파일_팀왈도폴아웃4.txt
마스터파일_폴런던기계번역문장.txt
그냥 메모장으로 열어보면 한 방에 감 온다.

[4.마스터파일을XML로변환(한글만).bat] 클릭하면 마스터 파일을 다시 XML로 만들어 준다.
혹은
[5.마스터파일을XML로변환(영문병기).bat] 클릭하면 마스터 파일을 다시 영문 병기된 XML로 만들어 준다.

메인소스_팀왈도폴아웃4.txt 은 영문병기 안했다...검증된 파일이라

그럼 이걸 다시 xTranslator에 Import XML 하면 끝~~

XML Import 창에서 Overwrite Everything. Mode Use Strict FormID + Strings 선택해줄것

그러면 상단에 String 파일 참조 스트링들이랑 일부 문구는 깨질텐데 무시해도됨...

그외 [3.손번역패치를마스터파일로병합.bat] 클릭하면 

마스터파일_손번역패치.txt 파일을 따로 만들어서 작업할 경우 마스터 파일에 자동으로 병합시켜준다. 그냥 혼자서 하는 경우 마스터 파일만 가지고 해도 무관

완전 오픈 소스니깐 안심하고 사용해도 됨...
